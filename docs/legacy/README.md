# Legacy

Este directorio contiene materiales valiosos pero no canónicos.

Razones típicas para que algo viva aquí:

- contiene razonamiento interno o trazas de conversación,
- mezcla texto final con basura de ejecución,
- requiere limpieza antes de entrar a `canon/`,
- documenta una etapa histórica útil pero no estable.
